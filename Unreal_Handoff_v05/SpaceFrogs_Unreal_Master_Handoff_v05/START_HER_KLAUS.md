# START HER — SpaceFrogs Unreal Master Handoff v05

Denne pakke samler:
- Content/FROG_GAME bibliotek-struktur
- Sector 7F concept GLB
- greybox OBJ-moduler
- Windows installer

## Brug

1. Pak ZIP ud.
2. Dobbeltklik `INSTALL_SPACEFROGS_MASTER_TO_UNREAL_WINDOWS.bat`.
3. Indsæt din Unreal project folder, fx:
`C:\Users\x\Documents\Unreal Projects\FROG3D_v01`

## I Unreal

Gå til:
`Content/FROG_GAME/Sector7F`

Lav map:
`LVL_Sector7F_BioforgeBreach`

GLB = reference.
OBJ modules = gameplay blockout.
